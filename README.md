## Особенности

- **Два языка** — русский (`/ru`) и английский (`/en`) с переключателем в шапке
- **Адаптивная вёрстка** — корректное отображение на телефонах и планшетах
- **Тёмная тема** и анимации (в т.ч. снег)
- **Скачивание игры** — кнопка с индикатором прогресса (URL настраивается через переменную окружения)
- **Навигация** — логотип, меню (Главная, Магазин, Правила, Discord), мобильное меню и свайпы
- **Магазин** — редирект на Discord для покупок
- **Правовые страницы** — правила игры, политика конфиденциальности, пользовательское соглашение (на обоих языках)
- **SEO** — метаданные и альтернативные ссылки для ru/en

## Установка и запуск

### 1. Клонирование

```bash
git clone https://github.com/ZAREZOV2/mytanks.git
cd mytanks
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Переменные окружения (опционально)

Создайте файл `.env.local` в корне:

```env
# URL для скачивания игры
NEXT_PUBLIC_GAME_URL=https://github.com/ZAREZOV2/mytanks/releases/latest/download/game.zip
```

### 4. Запуск в режиме разработки

```bash
npm run dev
```

Сайт будет доступен по адресу [http://localhost:3000](http://localhost:3000). Корневой путь `/` перенаправляется на `/ru` (локаль по умолчанию).

## Структура проекта

```
mytanks/
├── app/
│   ├── [locale]/                 # Маршруты с локалью (ru, en)
│   │   ├── components/           # Header, SnowEffect
│   │   ├── donate/               # Магазин
│   │   ├── privacy/              # Политика конфиденциальности
│   │   ├── rules/                # Правила игры
│   │   ├── terms/                # Пользовательское соглашение
│   │   ├── layout.tsx            # Layout с метаданными и провайдером переводов
│   │   ├── page.tsx              # Главная страница
│   │   └── page.module.css
│   ├── api/                      # API (например, discord-news)
│   ├── assets/                   # Изображения для импорта
│   ├── globals.css
│   └── layout.tsx                # Корневой layout
├── i18n/
│   ├── request.ts                # Загрузка сообщений для next-intl
│   └── routing.ts                # Локали (ru, en), префикс в URL
├── messages/
│   ├── ru.json                   # Тексты на русском
│   └── en.json                   # Тексты на английском
├── public/
│   └── assets/                   # Публичные ресурсы (логотип, фон)
├── i18n.ts                       # Константы локалей
├── middleware.ts                 # next-intl: редирект / → /ru, определение локали
├── next.config.js
├── package.json
└── tsconfig.json
```

## Языки и локализация

- **Русский** — пути вида `/ru`, `/ru/rules`, `/ru/donate` и т.д.
- **Английский** — пути вида `/en`, `/en/rules`, `/en/donate` и т.д.
- Переключатель **RU | EN** в шапке сохраняет текущую страницу при смене языка.
- Тексты задаются в `messages/ru.json` и `messages/en.json`; структура ключей совпадает в обоих файлах.
- Метаданные (title, description, Open Graph) и даты на страницах зависят от выбранной локали.

## Деплой

### Vercel (рекомендуется)

1. Подключите репозиторий [ZAREZOV2/mytanks](https://github.com/ZAREZOV2/mytanks) к Vercel.
2. При необходимости задайте переменную окружения `NEXT_PUBLIC_GAME_URL` в настройках проекта.
3. В разделе **Domains** добавьте домен `mytanks.online` и настройте DNS по инструкциям Vercel.

### Vercel CLI

```bash
npm i -g vercel
vercel login
vercel
# для production:
vercel --prod
```
